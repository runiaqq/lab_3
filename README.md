# Лабораторна робота №3 (NumPy)

**Тема:** Бібліотека NumPy. Створення і опрацювання багатовимірних масивів.

**Завдання:** сформувати 2 матриці \(3 \times 4\) випадкових чисел у діапазоні **(-5..6)**, сформувати матрицю з **парних стовпчиків** (2-й та 4-й) вхідних матриць, вивести підмасиви \(2 \times 2\) з:
- нижнього правого кута матриці **A**
- верхнього лівого кута матриці **B**

## Встановлення залежностей

```bash
python -m pip install -r .\requirements.txt
```

## Запуск

```bash
python .\main.py
python .\main.py 1
```

`seed` можна передати аргументом командного рядка, щоб результат був відтворюваним.
