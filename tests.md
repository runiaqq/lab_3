# Тести для відлагодження

Програма детермінована: однаковий `seed` ⇒ однакові матриці і вивід.

## Тест 1 (seed за замовчуванням)

```bash
python .\main.py
```

Перевірити:
- `A` і `B` мають розмір **3x4**, елементи в межах **-5..6**
- `C` складається з колонок **2 і 4** матриці `A` + колонок **2 і 4** матриці `B`
- підмасив `A` (2x2) — **нижній правий**
- підмасив `B` (2x2) — **верхній лівий**

## Тест 2 (інший seed)

```bash
python .\main.py 1
```

Очікування: числа зміняться, але всі властивості з Тесту 1 мають виконуватися.
